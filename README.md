<h2>Instalação 👨‍💻</h2>
Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:

- <b>[Git](https://git-scm.com)</b>

### E então:

```
- Clone o repositório:
$ git clone https://github.com/ggonpereira/search-google-clone.git

- Depois disso acesse o diretório:
$ cd search-google-clone

- Por fim, digite:
$ index.html
```
